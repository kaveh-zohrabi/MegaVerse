# MegaVerse Web Application

Next.js-based web application for the MegaVerse platform.

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State**: Zustand / React Query
- **Auth**: NextAuth.js
- **Testing**: Vitest + Playwright

## Features

- Server-side rendering
- Static site generation
- API routes
- Authentication
- Responsive design
- Dark mode
- i18n support
- PWA capabilities

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Testing

```bash
pnpm test
pnpm test:e2e
```
