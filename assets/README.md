# MegaVerse Static Assets

## Structure

```
assets/
├── images/
│   ├── logo/           # Brand logos
│   ├── icons/          # Application icons
│   ├── backgrounds/    # Background images
│   ├── avatars/        # Default avatars
│   └── screenshots/    # App screenshots
├── videos/
│   ├── tutorials/      # Tutorial videos
│   ├── demos/          # Demo videos
│   └── recordings/     # Recorded sessions
├── audio/
│   ├── sfx/            # Sound effects
│   └── music/          # Background music
├── fonts/
│   ├── web/            # Web fonts
│   └── desktop/        # Desktop fonts
├── documents/
│   ├── whitepapers/    # Technical whitepapers
│   ├── presentations/  # Slide decks
│   └── diagrams/       # Architecture diagrams
├── data/
│   ├── sample/         # Sample datasets
│   ├── benchmarks/     # Benchmark data
│   └── test-data/      # Test fixtures
└── models/
    ├── 3d/             # 3D models
    ├── icons/          # Icon SVGs
    └── sprites/        # Sprite sheets
```

## Guidelines

- Use SVG for icons and logos
- Use WebP for photos (with JPEG fallback)
- Optimize all images
- Use responsive images
- Follow accessibility guidelines (alt text)
