# MegaVerse Icons

SVG icon library for MegaVerse applications.

## Usage

```tsx
import { UserIcon, HomeIcon, SettingsIcon } from '@megaverse/icons';

function MyComponent() {
  return (
    <nav>
      <HomeIcon size={24} />
      <UserIcon size={24} />
      <SettingsIcon size={24} />
    </nav>
  );
}
```

## Available Icons

- Navigation: Home, Menu, ArrowLeft, ArrowRight
- User: User, UserPlus, UserMinus
- Actions: Edit, Delete, Plus, Minus, Check, X
- Communication: Mail, Phone, MessageSquare
- Media: Image, Video, Music
- Files: File, Folder, Download, Upload
- Social: Heart, Star, Share, Bookmark
