# MegaVerse UI Library

Shared React component library for all MegaVerse frontend applications.

## Components

- Buttons
- Forms
- Modals
- Cards
- Navigation
- Data tables
- Charts
- Notifications
- Tooltips
- Badges
- Avatars
- Icons

## Usage

```tsx
import { Button, Modal, Card } from '@megaverse/ui';

function MyComponent() {
  return (
    <Card>
      <Button variant="primary">Click me</Button>
    </Card>
  );
}
```
