# AI Service

Machine learning inference, embeddings, and AI-powered features.

## Features

- Text generation
- Text embeddings
- Image recognition
- Sentiment analysis
- Content moderation
- Recommendation engine
- RAG (Retrieval-Augmented Generation)
- Agent orchestration

## Endpoints

- `POST /ai/generate` - Generate text
- `POST /ai/embeddings` - Create embeddings
- `POST /ai/classify` - Classify content
- `POST /ai/moderate` - Moderate content
- `POST /ai/recommend` - Get recommendations
- `POST /ai/rag/query` - RAG query
- `POST /ai/agents/run` - Run agent
- `GET /ai/models` - List models
- `GET /ai/models/:id` - Get model info
