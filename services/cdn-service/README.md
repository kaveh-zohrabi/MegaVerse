# CDN Service

Content delivery and edge caching.

## Features

- Global edge network
- Static asset caching
- Dynamic content acceleration
- Image optimization
- Video delivery
- DDoS protection
- SSL termination
- Analytics

## Endpoints

- `GET /cdn/purge` - Purge cache
- `GET /cdn/stats` - CDN statistics
- `POST /cdn/configure` - Configure CDN
- `GET /cdn/health` - CDN health check
