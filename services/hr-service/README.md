# HR Service

Human Resources management.

## Features

- Employee profiles
- Time tracking
- Leave management
- Payroll integration
- Performance reviews
- Training tracking
- Org chart
- Document management

## Endpoints

- `GET /employees` - List employees
- `POST /employees` - Add employee
- `GET /employees/:id` - Get employee
- `PUT /employees/:id` - Update employee
- `GET /time-off` - List time off requests
- `POST /time-off` - Request time off
- `PUT /time-off/:id` - Approve/deny request
- `GET /org-chart` - Get org chart
