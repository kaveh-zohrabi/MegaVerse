# Search Service

Full-text search and discovery across the platform.

## Features

- Full-text search
- Faceted search
- Autocomplete
- Search suggestions
- Indexing pipeline
- Multi-language support
- Search analytics
- Custom ranking

## Endpoints

- `GET /search` - Search
- `GET /search/autocomplete` - Autocomplete
- `GET /search/suggestions` - Suggestions
- `POST /search/index` - Index content
- `DELETE /search/index/:id` - Remove from index
- `PUT /search/index/:id` - Update index
