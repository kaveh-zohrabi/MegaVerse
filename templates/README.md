# MegaVerse Project Templates

## Overview

Ready-to-use templates for new projects.

## Service Templates

### Go Service
```bash
cp -r templates/service/go services/my-new-service
```

### Java Service
```bash
cp -r templates/service/java services/my-new-service
```

### Node.js Service
```bash
cp -r templates/service/node services/my-new-service
```

### Python Service
```bash
cp -r templates/service/python services/my-new-service
```

### Rust Service
```bash
cp -r templates/service/rust services/my-new-service
```

## Library Templates

cp -r templates/library/typescript libraries/my-new-lib

### Go Library

cp -r templates/library/go libraries/my-new-lib
